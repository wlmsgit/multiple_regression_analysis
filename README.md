# Multiple Linear Regression – Multi-Channel Marketing Analysis

This project analyzes a marketing dataset to identify how TV, Radio, and 
Social Media advertising spend collectively impact Sales using Multiple 
Linear Regression with statsmodels OLS.

## What it does
- Loads and cleans a marketing spend dataset
- Checks for multicollinearity using correlation matrix and VIF
- Builds and validates a Multiple Linear Regression model
- Interprets Adjusted R-squared, coefficients, and p-values
- Validates assumptions with residual diagnostic plots
- Recommends budget allocation based on multi-channel ROI analysis

## Setup
pip install pandas numpy matplotlib statsmodels scikit-learn

## Usage
Run all cells in multiple_regression_analysis.ipynb top to bottom.

## Dataset
Simulated marketing dataset with 200 records across TV, Radio, 
Social Media spend and Sales variables.
